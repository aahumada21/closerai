# Workflows

Carpeta para artefactos relacionados a workflows de n8n.

- `workflows/exports/`: exports JSON (versionables) de workflows.
- `workflows/modules/`: módulos (sub-workflows) por responsabilidad.
- `workflows/examples/`: ejemplos pequeños o de referencia.
- `workflows/catalog/`: inventario y metadatos (source of truth local para Codex).

Nota:
- El sync guarda exports solo en `workflows/exports/uncategorized/` para no depender de tags.
