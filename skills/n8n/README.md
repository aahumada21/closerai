# Skills de n8n

Coloca aquí los “skills”/patrones reutilizables que el agente debe usar para construir workflows.

## Instalado
Este repo incluye una copia (vendorizada) de `czlonkowski/n8n-skills` en:
- `skills/n8n/n8n-skills/`

Recomendado:
- Un skill por carpeta: `skills/n8n/<skill_name>/`
- Cada skill con:
  - `SKILL.md` (qué hace, cuándo usarlo, inputs/outputs)
  - `examples/` (payloads y resultados esperados)
  - `snippets/` (bloques de nodos/expresiones)

Cuando me los compartas, actualizaré `RULE.md` para referenciar nombres reales y cómo invocarlos.
